Aplicativo de lista de cartão de visita onde é possível manter os dados nome, empresa, telefone de contato, email e cor de fundo do cartão utilizando banco de dados Room.

## <br />Tecnologias utilizadas
1. Room
2. MVVM
3. ViewBinding
4. LiveData
5. ListAdapter
6. Material Design
